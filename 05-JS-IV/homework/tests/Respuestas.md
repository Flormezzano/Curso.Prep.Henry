
* Un objeto es un tipo de dato de Javascript. Es una colección de propiedades. Dentro de los objetos se pueden almacenar todo tipo de datos. Se puede decir que todo en JavaScript son objetos.  

* Propiedades Una propiedad es una asociación par entre una clave y un valor; esto se denomina keyword.    

* Métodos se le denomina a las funciones dentro de los objetos

* Bucle `for…in` Sirve para iterar osbre cada par clave-valor en un objeto, como los opbjetos no tienen indice numérico, el bulce for estándar no funciona. for ... in es una sintaxis algo diferente que recorrerá cada clave y termina cuando itera todas las claves.

* Notación de puntos vs notación de corchetes. Para acceder a las propiedades o para asignar valores podemos hacerlo de dos formas; con la notacion de puntos se puede llamar al nombre del objeto un punto y el nombre de la clave.
con la notación de corchetes es como llamar a un elemento en un array aunque al usar corchetes se debe envolver a cada clave entre comillas. ['variable']
