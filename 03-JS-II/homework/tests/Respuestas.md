
* `for` Se usa cuando se quiere repetir una tarea o instrucciones una cierta cantidad de veces sin tener que hacerlo una por una de forma manual. En estos casos también se puede usar while; este es algo más manual, pero sirve cuando no se sabe la condición de parada, es decir hasta donde debería ejecutar. En cambio cuando se usa for de antemano se configura donde deberia detenere la función.

* `&&`, `||`, `!` Estos símbolos se llaman operadores de lógica; son como operadores matemáticos pero para tipos de datos booleanos. Un operador recibe dos operandos y los compara para luego devolver una respuesta segun corresponda. 
'And' se representa con &&, esta evalúa las dos expresiones y devuelve verdadero solo si ambas son verdaderas. 'Or' se representa con ||, de la misma forma que la anterior, evalua dos expresiones y retornará verdadero si una o ambas son verdaderas, devolverá false si ambas son falsas.
y por ultimo 'Not' que se representa con !, es una negación ante lo indicado, devolverá el booleano opuesto; false es false, pero !false nos devolverá true.
